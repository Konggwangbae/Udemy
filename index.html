HTML & CSS Summary

HTML Summary

HTML (HyperText Markop Language) is used ro define our page content, structure and meaning. You don't use it for sryling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag, content and then a closing tag
  • You can also have void (empty) elements like images
  • You can also configure elements with attributes
  • There's a long list of available elemnts but you'll gain experience over time, no worries.

learn more abuot all available HTML elements on the MDN HTML elements reference.

CSS Summary

CSS (Casecading style stylesheets) is used for styling your page content.

  • styles are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code duplication, you typically use global styles (e.g. via the "style" element) instead
  • Alternatively, you can work external stylesheet files which you "link" to
  • when working with CSS, conxepts like "inheritance", "specificity" and the "box model" should be understood.

learn more abuot all available CSS properties and values on the MDN CSS property reference.
